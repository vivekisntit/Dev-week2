# Grade Calculator
## Project overview
The Student Grade Calculator is a Python program that takes a student’s name and marks as input, calculates the respective grade (A, B, C, D, F) and displays a message based on performance.

This project demonstrates fundamental Python concepts including conditional statements, loops, functions, and input validation.

## Technical Requirements Implemented

- Uses if-elif-else statements for grading logic
- Includes input validation (inputs marks must be between 0–100)
- Displays a message for each grade
- Uses input() to collect user data
- Uses print() for formatted output
